01 — Sobre o projeto

Uma arquitetura pensada para escalar

Este projeto foi desenvolvido utilizando uma Arquitetura de Microsserviços. Em vez de um sistema monolítico, a aplicação foi dividida em serviços menores e independentes — como os microsserviços de Usuários, Catálogo, Reservas, Relatórios e Empréstimos — que se comunicam de forma eficiente entre si. Essa abordagem garante maior escalabilidade, resiliência e facilidade na manutenção do código.

Tecnologias utilizadas

React 19 TypeScript Vite Tailwind CSS Node.js Prisma ORM RabbitMQ Docker AWS Jenkins
