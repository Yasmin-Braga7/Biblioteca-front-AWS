📚 Sistema de Biblioteca

Plataforma web para gestão de acervo, usuários, empréstimos e reservas.

Sobre o Projeto

Este projeto foi desenvolvido utilizando uma Arquitetura de Microsserviços. Em vez de um sistema monolítico, a aplicação foi dividida em serviços menores e independentes, como os microsserviços de Usuários e Empréstimos (entre outros), que se comunicam de forma eficiente. Isso garante maior escalabilidade, resiliência e facilidade na manutenção do código.

🛠️ Tecnologias Utilizadas

O ecossistema do projeto (Front-end e Back-end) foi construído utilizando:

React / Vite Node.js Prisma ORM RabbitMQ AWS Docker
